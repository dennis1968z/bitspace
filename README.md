# BitSpace 公司网站

Oracle (甲骨文 OCI) 风格的企业官网。静态站点，无需构建——双击 `index.html` 即可在浏览器打开。

## 公司定位
- **BitSpace** — Oracle Cloud Infrastructure (OCI) 亚太区合作伙伴
- 两条业务线：①基于 OCI 的云 / 算力 / 存储基础服务；②OPC OS 一人公司操作系统
- 两地运营：新加坡（亚太总部 / 主节点）& 硅谷（研发中心）

## 页面
| 文件 | 内容 |
|------|------|
| `index.html` | 首页：愿景、两条业务线、平台架构总览、价值主张、模型路由 |
| `cloud.html` | 云基础设施：核心服务、为什么选 OCI、区域节点、合规、技术栈 |
| `opc.html` | OPC OS 平台：价值对比、Agent Matrix、工作循环、5 大垂直服务、定价、集成 |
| `about.html` | 关于：愿景、使命、全球布局、护城河、团队 |
| `contact.html` | 联系 / 预约演示表单（演示用，未实际发送） |

## 设计
- 配色参考甲骨文：Oracle Red `#c74634` + 墨黑 + 暖灰
- `assets/css/style.css` 全站样式；`assets/js/main.js` 移动端导航 + 数字滚动动画
- 响应式，适配桌面 / 平板 / 手机

## 内容来源
所有业务数据（定价、毛利、架构、指标、融资）均提取自 `OPC_BP_v3.pdf` 商业计划书及架构总览图。
