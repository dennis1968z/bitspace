关于这份目录

以上为 OPC OS 推荐的 Agent 角色，并非固定清单——平台支持按你的垂直场景动态招募与自定义 Agent。每个 Agent 通过统一的 Credential Vault 加密访问外部工具，并受权限沙箱约束，你可精细控制每个 Agent 对每个工具的读/写权限。

所有 Agent 都在 Think → Do → Check → Improve 工作循环中协作，由 智能模型路由 在 L1–L4 之间为每个任务选择最合适的模型。
